# General Recommender System

## 不同的推荐场景

### 1. 短视频场景
高频反馈、强调留存与沉浸时长，收益常来自广告与直播。

### 2. 电商场景
全链路行为明显，搜索意图强，核心指标常围绕 GMV/CVR。

### 3. 内容分享/种草社区
收藏/评论等信号价值高，需兼顾社区调性与转化价值。

### 4. 长视频/流媒体
关注会员价值（LTV、续费率）与内容利用效率。
